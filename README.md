# The-meta-you-need-to-know

> **Meta** 在 HTML 中大家都知道是用来描述页面信息的，但是具体有哪些呢？各自的使用方式及参数是什么呢，或许跟我现在一样 ~懵逼~

### 前言

在说 meta 之前我们先来了解下 `Metadata`，翻译过来是 **元数据**，是描述数据的数据(data about data)，用于描述数据属性（property）的信息，_在 HTML 中就是描述页面属性的一些描述信息_
